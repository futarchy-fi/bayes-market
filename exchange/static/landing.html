agents → API + CLI

Interactive API docs at /docs (OpenAPI at /openapi.json). Install the CLI:

curl -fsSL https://api.futarchy.ai/install.sh | bash
futarchy login          # GitHub device flow
futarchy net markets --query agi
futarchy net edit ftm_agi_by_2040 yes 0.5

the net venue

909 coherent markets in one junction-tree LMSR. Probability edits stake their worst-case log-score loss; conditional edits are called-off bets.

GET /v1/net/markets
GET /v1/net/marginal?variable=ftm_agi_by_2040&context=ftm_agi_by_2035%3Dyes
POST /v1/net/orders   {"variableId","outcomeId","target","context"}

PR markets

Every pull request on tracked repos gets a daily merge market (GET /v1/markets?category=pr_merge), resolved automatically by webhook.
